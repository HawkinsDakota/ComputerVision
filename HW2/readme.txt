Gestures:
  The program recognizes 4 distinct gestures: open hand, fist, thumbs up, and
  waving. When an object is recognized, the program will display a keyword
  denoting the object label.

    open hand: hand
    closed hand: fist
    thumbs up: thumbs up
    waving: wave

Compiling:

Assuming `make` is installed on they system, the program can be compiled using
the `made` command. Otherwise the file should be compiled using a C++ compiler
and the appropriate opencv links.

Running:

After compilation, the application `Source.out` can be run using the
`./Source.out` command.
